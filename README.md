# PDF Test

This small script serves the `assets` folder on localhost and opens the PDF in your default browser.

Run using the project's virtual environment Python:

```powershell
"c:/Users/Tauno/OneDrive/Desktop/Touno potfolio/.venv/Scripts/python.exe" test_open_pdf.py
```

The server will run briefly and then stop automatically after 15 seconds. You can increase the sleep time in `test_open_pdf.py` if needed.
